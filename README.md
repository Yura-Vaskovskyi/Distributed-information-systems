# TCP Sockets

Програма повинна коректно обробляти наступні команди:

1. ping

Не має параметрів

Використовується для перевірки з'єднання з сервером


2. echo <text>

Приймає текстовий параметр

Відправляє текст на сервер та отримує відповідь


3. login <login> <password>

Приймає логін і пароль

Виконує вхід або реєстрацію користувача


4. list

Не має параметрів

Отримує список активних користувачів із сервера


5. msg <destinationUser> <text>

Приймає ім’я користувача та текст повідомлення

Надсилає повідомлення іншому користувачу


6. file <destinationUser> <filename>

Приймає ім’я користувача та назву файлу

Надсилає файл іншому користувачу


7. exit

Завершує виконання програми

Коректно закриває TCP-з'єднання з сервером
