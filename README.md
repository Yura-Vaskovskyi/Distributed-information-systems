# Message Queue
Дана програма реалізує консольний чат-клієнт, який використовує технологію черг повідомлень для обміну даними між користувачами. Взаємодія з сервером здійснюється через систему повідомлень, що забезпечує надійну доставку та асинхронну обробку даних.

Програма повинна коректно обробляти наступні команди:

1. ping

Не має параметрів

Використовується для перевірки з'єднання через брокер повідомлень (ActiveMQ)


2. echo <text>

Приймає текстовий параметр

Відправляє повідомлення через чергу та отримує відповідь


3. login <login> <password>

Приймає логін і пароль

Виконує вхід або реєстрацію користувача


4. list

Не має параметрів

Отримує список активних користувачів


5. msg <destinationUser> <text>

Приймає ім’я користувача та текст повідомлення

Надсилає повідомлення іншому користувачу через чергу


6. file <destinationUser> <filename>

Приймає ім’я користувача та назву файлу

Надсилає файл іншому користувачу через чергу


7. exit

Завершує виконання програми

Коректно закриває з'єднання з брокером повідомлень
