# Remote Method Invocation

Програма повинна коректно обробляти наступні команди:

1. ping

Не має параметрів

Використовується для перевірки з'єднання з RMI сервером


2. echo <text>

Приймає текстовий параметр

Викликає віддалений метод та отримує той самий текст у відповідь


3. login <login> <password>

Приймає логін і пароль

Виконує вхід або реєстрацію користувача та отримує session id


4. list

Не має параметрів

Отримує список активних користувачів із сервера


5. msg <destinationUser> <text>

Приймає ім’я користувача та текст повідомлення

Надсилає повідомлення іншому користувачу через RMI


6. file <destinationUser> <filename>

Приймає ім’я користувача та назву файлу

Надсилає файл іншому користувачу через RMI


7. exit

Завершує виконання програми

Коректно завершує роботу клієнта
